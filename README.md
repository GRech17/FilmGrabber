## Description 

  *The what, why, and how:* 
  
  ğŸ” Film Grabber is an application where users can view trending movies, search for movies, see movie details and create a watchlist of movies they want to see. Some of the information returned from the search includes the movie title, overview, popularity, revenue and release date. This application was made to help those movie fanatics pick the right movie at home during COVID-19.

## Installation
navigate to desired installation directory and run "git clone https://github.com/GRech17/film-grabber.git".  After that, while in the root directory of the project inside the terminal, run "npm i" to install all node packages needed for the application.
